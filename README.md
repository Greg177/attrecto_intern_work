## Funkciók

- React alkalmazás TypeScript használatával
- Újrafelhasználható komponensek kialakítása
- Counter funkció state managementtel
- Oldalak közti navigáció React Router segítségével
- Filmek listázása statikus adatforrásból
- Filmek törlése a listából
- Film címének szerkesztése inline módon
- Üres mentés tiltása
- Aktívan szerkesztett sor kiemelése
- Lista rendezése:
  - cím (alphabetical)
  - értékelés (rating)
  - megjelenési év (year)
- Rendezési irány váltása (növekvő / csökkenő)
- Alapértelmezett rendezés: értékelés szerint csökkenő
